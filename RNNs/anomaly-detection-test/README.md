# anomaly-detection
Anomaly Detection Service

Anomaly Detection is a binary classification problem- given a data point in time containing values for certain pre-selected parameters, classify that point as an anomaly or not.

PROJECT FILES DETAILS:

1. tutorials/experiments: code used only for learning purposes. Please ignore.

2. data folder:
    - contains csv files for all fields.

    - subfolder- aggregate data:
        - contains plots, csv file for different fields combined into one.
        - plots for different parameters in the plots folder

3. kera_neural_networks:
    - see lstm.py for the rnn implementation.
